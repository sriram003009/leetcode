## 🟢 Easy

### 🔹 Reverse a Linked List

![Linked List](https://img.shields.io/badge/Pattern-Linked%20List-blue)

**Problem:**
Given the head of a singly linked list, reverse the list and return the reversed list.

---

### 🔹 Detect Cycle in a Linked List

![Linked List](https://img.shields.io/badge/Pattern-Fast%20%26%20Slow%20Pointers-blue)

**Problem:**
Given `head`, determine if the linked list has a cycle.

**Note:**
A cycle exists if a node can be revisited by following `next` pointers.

## 🔴 Hard

### 🔹 Minimum Window Substring

![Sliding Window](https://img.shields.io/badge/Pattern-Sliding%20Window-red)

Find the smallest substring of `s` that contains all characters of `t`.

**Examples:**

```
Input: s = "ADOBECODEBANC", t = "ABC"
Output: "BANC"

Input: s = "a", t = "aa"
Output: ""
```
